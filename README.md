# SANDRAY

SANDRAY is a handheld voice-operated AI assistant designed for the ClockworkPi uConsole.

## Features

- Voice input using Whisper.cpp
- AI conversation using AIChat
- Speech synthesis using Piper
- Rich terminal interface
- Configurable behaviour via YAML
- Modular architecture
- Git version control
